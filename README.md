# PixelNorm CDN

Official CDN repository for PixelNorm template enhancements.

PixelNorm provides lightweight design upgrades, theme systems, and interaction layers for website platforms including Squarespace.

## Available Modules

### SPARK

Dark / Light mode enhancement system.

Features:

- Automatic dark theme conversion
- Persistent visitor preference
- Header integration
- Mobile menu support
- Navigation and dropdown styling
- Image-safe rendering
- Footer support


## Installation

Add the following code inside your website header injection:

```html
<link rel="stylesheet"
href="https://cdn.jsdelivr.net/gh/pixelnorm/pixelnorm-cdn/spark/latest/spark.css">

<script src="https://cdn.jsdelivr.net/gh/pixelnorm/pixelnorm-cdn/spark/latest/spark.js"></script>
